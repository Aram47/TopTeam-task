# Nordisk Dekor - Landing Page

Single-page website (landing) for an interior decor and styling company in Denmark.

## Project Structure

```
TopTeam/
├── index.html                    # Main page
├── styles.css                    # Main styles
├── script.js                     # JavaScript functionality
├── pages/
│   ├── privacy-policy.html      # Privacy Policy
│   ├── terms-of-use.html        # Terms of Use
│   └── cookies-policy.html      # Cookies Policy
├── images/
│   ├── favicon.ico              # Favicon
│   ├── logo.svg (или .png)      # Logo (without text)
│   ├── hero-image.jpg           # Main image
│   ├── gallery-1.jpg            # Gallery image 1
│   ├── gallery-2.jpg            # Gallery image 2
│   └── gallery-3.jpg            # Gallery image 3
├── robots.txt                   # File for search engines
├── sitemap.xml                  # Sitemap
└── README.md                    # This file
```

## Technologies

- **HTML5** - semantic markup
- **CSS3** - responsive styles with CSS variables
- **Vanilla JavaScript** - no dependencies
- **Mobile-first** design approach

## Features

- ✅ Fully responsive design (mobile, tablet, desktop)
- ✅ Scandinavian minimalist style
- ✅ Smooth anchor scrolling
- ✅ Mobile menu (hamburger)
- ✅ Contact form validation
- ✅ Scroll animations
- ✅ Accessibility (ARIA, keyboard navigation)
- ✅ SEO optimization (meta tags, semantic HTML)
- ✅ All text in Danish

## Page Structure

1. **Hero Section** - main banner with call-to-action
2. **Services** - description of services (4 cards)
3. **Gallery** - portfolio gallery (3 projects)
4. **Features** - company advantages (4 items)
5. **Testimonials** - client reviews (3 reviews)
6. **Contact** - contact form and information

## Installation and Running

1. Clone or download the project
2. Make sure all images are in the `images/` folder
3. Open `index.html` in your browser
4. For a local server you can use:

   ```bash
   # Python 3
   python -m http.server 8000

   # Node.js (с http-server)
   npx http-server
   ```
